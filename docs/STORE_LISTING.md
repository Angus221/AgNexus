# AG Nexus - Edge 商店发布信息

## 扩展名称
AG Nexus

## 简短描述（132字符以内）
智能生产力中枢 - 集成导航、指令、提示词、待办、密码保险库和 AI 助理，一站式提升工作效率

## 详细描述

### AG Nexus - 你的智能生产力中枢

AG Nexus 是一款功能强大的浏览器侧边栏扩展，将多种实用工具整合在一起，帮助你更高效地完成日常工作。

### 核心功能

**快捷导航**
- 一键保存常用网站
- 自动获取网站图标
- 快速访问，告别繁琐的书签管理

**指令中心**
- 存储常用代码片段、命令行指令
- 一键复制，提升开发效率
- 支持任意文本内容保存

**提示词库**
- AI 提示词模板管理
- 标签分类，快速检索
- 复制即用，告别重复编写

**智能待办**
- 创建日常任务和提醒
- 支持日期、时间、优先级设置
- 悬浮球实时提醒，不错过重要事项
- 桌面通知，提前 5 分钟提醒

**密码保险库**
- PIN 码保护的安全存储空间
- 存储账号密码、API 密钥等敏感信息
- 本地加密，数据不上传云端

**AI 智能助理**
- 接入 OpenAI 兼容接口（默认阿里百炼）
- 自然语言交互，一句话完成操作
- 智能记忆系统，记住你的偏好
- 支持对话管理导航、待办等功能

### 特色亮点

- **侧边栏设计**：不遮挡网页内容，随用随开
- **悬浮球提醒**：页面角落实时显示待办状态
- **数据本地化**：所有数据存储在本地，保护隐私
- **导入导出**：支持数据备份和迁移
- **深色模式**：自动跟随系统主题
- **开源透明**：代码完全开源，接受社区监督

### 使用场景

- **开发者**：管理代码片段、API 文档、开发工具链接
- **内容创作者**：收藏 AI 提示词、管理创作素材
- **职场人士**：记录待办事项、保存常用网站
- **任何需要提升效率的用户**

### 隐私承诺

- 所有数据存储在本地浏览器中
- 不收集、不上传、不分享用户数据
- AI 对话仅发送至用户自己配置的 API 服务
- 完全开源，代码可审查

### 开始使用

1. 安装扩展后，点击工具栏图标打开侧边栏
2. 前往设置页面配置 AI API 密钥（可选）
3. 开始添加导航、创建待办、使用 AI 助理
